# Artusi Landing

Landing page for the Artusi client project by Q4 Studio.

## Overview
A lightweight, static marketing site built with HTML, CSS, and JavaScript.

## Project Structure
- `index.html` — main page
- `style.css` — styles
- `assets/` — images and media

## Local Development
Open `index.html` in your browser (or use a simple static server).
